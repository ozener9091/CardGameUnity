# Принцип работы игры

После нажатия кнопки **“Играть”** у каждого игрока генерируется собственная колода из 64 карт.

## Игровой процесс

- **Рука игрока:** 
  - Карты снизу и сверху представляют собой "руку игрока".
  - Автоматически в руку каждому игроку набирается по 4 карты. 
  - После каждого выбрасывания карты из руки на стол, в руку из колоды автоматически добираются карты, чтобы всегда было 4 карты.

- **Показатели карт:**
  - У каждой карты есть 2 показателя:
    - **Здоровье** (помечено зеленым цветом)
    - **Урон** (помечен красным цветом)

- **Основная карта:**
  - У каждого игрока есть основная карта, у которой в начале игры 30 очков здоровья и 1 очко урона.

## Цель игры

Главная задача игры — уничтожить основную карту соперника.

## Механика атаки

- Чтобы карта могла атаковать, её нужно положить на стол. 
- Для этого необходимо нажать по ней левой кнопкой мыши.
- За один ход из руки можно положить только 1 карту.

- После того как карта выложена на стол, у неё появляется индикатор атаки (в правом верхнем углу):
  - **Зеленый цвет** — карта может атаковать.
  - **Красный цвет** — карта не может атаковать.
